# Harnessing Disagreement: Detecting Correlated Agreement Blindness in Multi-Agent Triage

**Authors:** Shay Seiya McDonnell, Avantika Singh, Quoc-Viet Pham, Vratislav Havlik, Gregory M. P. O'Hare
**Published:** 2026-07-22 | **arXiv:** [2607.19899v1](http://arxiv.org/abs/2607.19899v1)

## Summary
This paper identifies a critical flaw in multi-agent safety systems: when independent agents improve and converge toward similar decisions, they become blind to correlated failures—situations where multiple agents make the same wrong call. The authors propose ARAT, a system using diverse reasoning models (Random Forest, k-NN, and a calibrated meta-model) that actively detects and escalates these dangerous agreement moments, validated on network intrusion detection and clinical readmission prediction tasks.

## Theoretical Background
Multi-agent systems often use disagreement-triggered escalation as a safety mechanism: when agents diverge, humans or oversight systems intervene. This assumes that disagreement correlates with mistakes. However, this approach builds on classical ensemble learning theory—the idea that combining diverse models reduces errors. The paper challenges a hidden assumption: as base learners improve (become more accurate), they naturally converge in their predictions, which weakens the disagreement signal. The authors also draw on calibration methods, which adjust model confidence scores to reflect true uncertainty, and case-based reasoning (k-NN analogy), which complements inductive learning (Random Forest).

## Core Contributions
1. **Identification of correlated agreement blindness**: Formalizes a structural vulnerability where safer, more accurate agents paradoxically create worse safety outcomes by agreeing on dangerous mistakes more often.
2. **Empirical quantification on real data**: Shows on UNSW-NB15 (82K samples) that 57.2% of errors occur under agreement, and 90.6% of dangerous under-predictions evade disagreement-based monitoring—a concrete measure of the problem's severity.
3. **Architectural mitigation via diversity and calibration**: ARAT combines inductive (RF) and analogical (k-NN) reasoning with a calibrated meta-model to recover productive disagreement. Reduces under-prediction errors from 4.80% to 1.70% via conservative override and safety-flag gating.
4. **Cross-domain validation**: Demonstrates the pattern holds beyond network security (intrusion detection) into healthcare (clinical readmission), suggesting the insight generalizes.

## Practical Applications
1. **Intrusion detection and network security**: Hospitals, financial institutions, and critical infrastructure can deploy ARAT-like systems to catch coordinated failure modes in anomaly detection, improving false-negative rates (missed attacks).
2. **Clinical decision support**: Reduce dangerous under-predictions in readmission risk, sepsis prediction, or diagnosis screening where correlated model agreement masks errors.
3. **Autonomous vehicle safety**: Multi-sensor or multi-model perception systems (LiDAR + camera + radar) can detect when all modalities agree on a dangerous misclassification (e.g., mistaking a pedestrian for road debris).
4. **Fraud detection in finance**: Credit card or AML systems using ensemble agents can catch scenarios where all models simultaneously approve a fraudulent transaction.
5. **Content moderation at scale**: Combine different reasoning approaches (rule-based, learned, case-based) to avoid systematic agreement on harmful content that evades disagreement-based human review queues.

## Technical Limitations
1. **Diversity by architecture alone**: ARAT uses RF and k-NN—fundamentally different learning paradigms. But in production systems, you may have only access to black-box API models or similar architectures, limiting the diversity required to generate productive disagreement.
2. **Calibration dependency**: The meta-model's safety depends on how well it calibrates uncertainty. In distribution shift (train on one network, test on another), calibration breaks, potentially re-creating blind spots.
3. **Computational overhead**: The system requires running multiple base learners plus a meta-model; for real-time systems (fraud, intrusion detection), this latency cost may be prohibitive.
4. **Limited scalability of k-NN**: k-NN's performance degrades as data dimensionality and volume increase, and retraining is expensive—problematic for high-dimensional sensor data or large-scale deployment.
5. **Generalization to correlated failure modes**: The paper tests on two domains (network + clinical). Correlated failure *causes* (e.g., dataset biases, adversarial examples) likely vary by domain; the approach may not transfer cleanly to domains with different failure correlations.
6. **Assumption that agreement can be measured reliably**: In soft-output systems (probabilities, scores), "agreement" requires threshold choices that may be arbitrary and sensitive.

## Open Questions
1. **Can productive disagreement be engineered?** Rather than hoping architectural diversity creates useful disagreement, can we train agents explicitly to disagree on hard cases? (E.g., adversarial training, curriculum learning on disagreement-inducing samples.)
2. **What is the fundamental limit of this approach?** Is there a regime where even diverse, well-calibrated agents must converge to the truth, making disagreement unavoidably rare and unhelpful?
3. **Does ARAT scale to LLM agents?** Modern agentic systems (e.g., multi-agent reasoning with LLMs) may be *more* correlated (all fine-tuned on similar data). Does the approach work there?
4. **How do we detect correlated agreement blindness *before* deployment?** Can we predict which domains or datasets are vulnerable to this failure mode using synthetic tests or theoretical measures?
5. **Can we trade off accuracy for disagreement-richness?** Would intentionally weakening base learners (lower individual accuracy) increase productive disagreement and overall safety, and at what cost?
6. **Adversarial robustness**: Do adversarial examples that fool one agent tend to fool all agents? If so, does ARAT remain blind to adversarial agreement failures?

## Key Takeaway
Safer AI agents that converge create hidden risks: they agree on dangerous mistakes, making disagreement-based safety monitoring blind—solved here with calibrated multi-method ensembles.
