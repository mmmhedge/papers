# SGD Provably Prioritizes a Shortcut Spurious Feature in the XOR Model

**Authors:** Tyler LaBonte, Vidya Muthukumar
**Published:** 2026-06-29 | **arXiv:** [2606.30444v1](http://arxiv.org/abs/2606.30444v1)

## Summary
This paper provides the first rigorous theoretical analysis of how stochastic gradient descent (SGD) learns spurious correlations in neural networks, using a two-layer ReLU network trained on XOR data with a linear spurious feature. The work reveals that SGD prioritizes the spurious shortcut feature and learns it exponentially fast, with precise phase transitions showing how spurious and signal features interact during optimization—a finding that has direct implications for understanding and mitigating shortcut learning in real models.

## Theoretical Background
The paper builds on long-standing concerns about neural networks' reliance on spurious correlations—patterns that predict labels in training data but don't reflect causal structure. Prior work has documented this empirically across vision and NLP, but lacks formal optimization-level explanations. The paper uses the XOR function as a canonical hard problem (nonlinearly separable on the Boolean hypercube) paired with a simpler linear spurious feature, creating a controlled setting where signal and shortcut features can be cleanly separated. The analysis leverages tools from implicit bias theory and neural network loss landscape analysis, studying how minibatch SGD on the logistic loss exhibits preference for different features during training.

## Core Contributions
1. **First end-to-end characterization**: Proves that SGD provably learns the spurious feature *before* the true XOR signal, with exponential speed, in a rigorous setting with concrete rates.
2. **Coupled dynamics analysis**: Shows how alignment between spurious feature signs and second-layer weights drives rapid spurious feature growth, while a strong spurious component actively *suppresses* signal learning through margin dynamics.
3. **Phase transition theory**: Identifies two distinct phases—an initial alignment-driven phase where spurious features explode, followed by a majority-group margin phase that slows learning and keeps the signal suppressed.
4. **Sample complexity threshold result**: When spurious correlation is maximally strong, proves the spurious feature dominates even at the sample size where XOR would be learned in isolation.

## Connections to Other Work
This work directly addresses a core question in [[mechanistic-interpretability]] and model robustness: *why* do networks learn shortcuts? It complements empirical work on shortcut learning and relates to broader implicit bias theory (e.g., margin maximization in linear models). The XOR setup connects to classical hardness results in learning theory, while the SGD dynamics analysis extends recent theoretical work on feature learning in neural networks. The paper also touches on alignment concerns in AI—understanding how models can learn unintended solutions is foundational to alignment research.

## Practical Applications
- **Dataset debugging**: Practitioners can use insights about spurious feature dominance to identify which correlations in their data SGD will likely exploit first, enabling targeted data augmentation or filtering.
- **Robustness evaluation**: The phase transition predictions allow validation of whether a model is in the "spurious-dominated regime" and at what sample size recovery becomes possible.
- **Training intervention timing**: The precise dynamics suggest when interventions (e.g., reweighting, early stopping) should be applied to prevent spurious feature lock-in.
- **Synthetic benchmarks**: Organizations building robustness benchmarks can use XOR-like constructions to isolate and test spurious feature learning in controlled settings.

## Product & Startup Ideas
1. **Spurious Feature Detector**: A tool that analyzes training dynamics to identify which features SGD is learning in which order, flagging likely shortcuts before deployment; targets ML teams building production models.
2. **Synthetic Data Curriculum Service**: Dynamically adjust data sampling or reweighting during training based on theoretical predictions of when spurious vs. signal features will dominate, optimized per architecture and dataset.
3. **Interpretability Dashboard for Model Auditing**: Real-time visualization of feature learning phase transitions in training, helping data scientists understand model behavior and catch shortcut learning before it becomes entrenched.
4. **Robustness Testing Suite**: Automated generation of XOR-like adversarial correlations in a client's dataset to test whether their model is vulnerable to spurious learning; measure "spurious feature dominance index."
5. **Adaptive Regularization System**: Automated meta-learner that detects when a model enters the margin-suppression phase and dynamically adjusts loss (e.g., via mixup or importance weighting) to promote signal feature learning.

## Technical Limitations
- **Restrictive setting**: Analysis only covers two-layer ReLU networks on Boolean hypercube data with quadratic (XOR) signal and linear spurious features. Real data and deeper networks have far more complex interactions.
- **Logistic loss assumption**: Results are specific to logistic loss; behavior under cross-entropy, MSE, or other losses remains uncharacterized.
- **Bounded dimensionality**: The high-dimensional regime assumed (dimension → ∞) may not capture finite-dimensional phenomena or inductive biases from data structure.
- **Minibatch SGD idealization**: Analysis assumes online minibatch SGD; practical optimizers (momentum, adaptive learning rates, batch norm) introduce unmodeled effects.
- **Single spurious feature**: Real data often has *multiple* spurious correlations with different strengths; interaction effects are unexplored.
- **Incomplete signal learning**: The paper provides only *preliminary empirical evidence* that the signal is eventually learned when spurious correlation strength is constant; this gap deserves closing.

## Open Questions
1. **Generalization to deeper networks**: Do the phase transitions and exponential spurious learning rates persist in deeper architectures? How do residual connections or normalization layers change the dynamics?
2. **Multiple spurious features**: What happens when data contains competing spurious features of different strengths? Which does SGD learn first, and can they interfere?
3. **Beyond XOR**: Can the analysis extend to other hard nonlinear problems (parity, threshold functions) or real-world synthetic datasets?
4. **Continuous correlation strength**: The paper assumes binary presence/absence of spurious correlation; what if spurious and signal features are *both* present as smooth functions? When does one dominate?
5. **Recovery mechanisms**: Under what conditions can a model unlearn a spurious feature once it has been learned? Can interventions provably accelerate signal learning in the suppressed regime?
6. **Algorithmic interventions**: Do the identified phase transitions suggest principled algorithmic defenses (e.g., adaptive learning rates, feature reweighting)? Can theory guide hyperparameter selection?

## Key Takeaway
SGD has a provable bias toward learning simpler spurious features before complex true signal, with coupled dynamics that actively suppress signal learning once spurious features dominate—a mechanism that persists even when data contains enough samples to solve the problem without shortcuts.
