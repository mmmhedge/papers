# MIRROR: Learning from the Other View for Multi-Modal Reasoning

**Authors:** Wen Ye, Yuxiao Qu, Aviral Kumar, Xuezhe Ma
**Published:** 2026-07-23 | **arXiv:** [2607.21552v1](http://arxiv.org/abs/2607.21552v1)

## Summary
This paper addresses a key weakness in vision-language models (VLMs): they often fail at visual reasoning tasks even when they could solve the same problem presented as text. The authors show that different representations of the same problem—text-only, image-only, or combined—expose different reasoning strengths and weaknesses. They propose MIRROR, a reinforcement learning method that trains models to learn from their best-performing modality, using the strongest view as a teacher for weaker ones, improving both accuracy and cross-modal consistency on geometry reasoning tasks.

## Theoretical Background
Vision-language models combine visual and textual processing through shared representations (often built on transformer — neural networks using attention mechanisms to weight different input parts). Large language models excel at reasoning but lack visual grounding; conversely, VLMs struggle with abstract reasoning despite having image understanding capabilities. The paper builds on self-supervised learning (training without manual labels by using model outputs as targets) and reinforcement learning with reward signals. It applies reverse-KL divergence — a statistical measure of how different two distributions are — as an optimization objective, where a weaker modality learns to match the probability distribution of the stronger one.

## Core Contributions
1. **ODA-Data dataset**: A curated geometry dataset with text-only, image-only, and combined views of identical problems, with explicit train/test splits for measuring modality-dependent behaviors.
2. **MIRROR algorithm**: A novel RL approach that selects the best-performing view per problem as a teacher and trains all other views via reverse-KL divergence toward that teacher, rather than using a fixed reward signal.
3. **Empirical evidence**: Demonstrates that modality inconsistency is systematic and exploitable—training models to learn cross-modal agreement improves both absolute accuracy and consistency across visual and textual representations.

## Connections to Other Work
This work relates to [[RLHF|reinforcement learning from human feedback]] and preference-based learning, but replaces human judgment with model-generated signals across modalities. It extends multimodal contrastive learning (e.g., [[CLIP]]) by explicitly modeling view-dependent reasoning. The reverse-KL objective connects to distillation approaches where a student model learns from a teacher. The paper implicitly addresses the broader problem of [[alignment]] between different processing pathways in neural networks—ensuring different representations converge on consistent outputs.

## Practical Applications
1. **Educational software**: Geometry tutoring systems that explain problems across multiple representations (diagram + text explanation), improving student understanding by showing complementary reasoning paths.
2. **Accessibility tools**: Auto-converting complex visual diagrams into text descriptions for visually impaired users, or vice versa, using models trained for cross-modal reasoning.
3. **Automated theorem proving**: Geometry theorem provers that handle both symbolic (text) and visual problem statements with consistent reasoning.
4. **Document understanding**: OCR and document analysis systems that reconcile extracted text with visual layout to resolve ambiguities.

## Product & Startup Ideas
1. **AI Geometry Tutor**: Interactive learning platform using MIRROR-trained models to teach geometry with automatic generation of equivalent text and visual explanations, adapting to student learning style.
2. **Document Intelligence Platform**: Enterprise solution that reconciles inconsistent OCR results with visual structure analysis using cross-modal reasoning to extract structured data from scanned documents.
3. **Visual Question Answering (VQA) API**: Commercial API for consistent, reliable visual reasoning (receipts, diagrams, charts) that maintains coherence across text extraction and visual interpretation.
4. **Accessibility Bridge**: Tool converting complex visual content (medical scans, engineering drawings, charts) into consistent, complementary text narratives and vice versa for users with different accessibility needs.
5. **Scientific Paper Analysis**: System that jointly reasons over paper figures, captions, and text to extract claims and evidence more reliably than text-only or image-only approaches.

## Technical Limitations
1. **Dataset scope**: ODA-Data is specific to geometry; unclear if MIRROR generalizes to other domains (medical imaging, charts, photographs) where visual and textual information have different information-theoretic roles.
2. **Computational cost**: Evaluating every problem under all views before training is expensive; scalability to large-scale data is unclear.
3. **Teacher selection**: Choosing the best-performing view assumes all views are equally valid ground truth; in some domains, one modality may be inherently more reliable, and forcing agreement could reduce overall accuracy.
4. **Assumption fragility**: The method assumes views have comparable expressiveness; fails if one modality genuinely cannot represent certain concepts.
5. **No theoretical guarantees**: No formal analysis of when/why the reverse-KL objective improves consistency or reasoning; improvement is purely empirical.

## Open Questions
1. Does MIRROR work on non-geometry domains (e.g., medical imaging, natural scene understanding, abstract diagrams)? How does modality complementarity vary by task?
2. What is the relationship between modality agreement and true accuracy? Can forced consistency degrade performance on domains where views legitimately encode different information?
3. How does MIRROR scale to trillion-parameter VLMs? Is the per-problem view selection bottleneck prohibitive?
4. Can the method be extended to >2 modalities (e.g., video + audio + text, or 3D spatial + 2D image + description)?
5. What happens when no single view dominates? How should the method select teachers in cases of tie or modest differences?
6. Does MIRROR transfer to downstream tasks (e.g., a model trained for consistency on geometry problems, when fine-tuned on new VQA tasks)?

## Key Takeaway
Different modalities of the same problem expose complementary reasoning strengths; teaching weaker modalities to match stronger ones improves multimodal reasoning consistency.
