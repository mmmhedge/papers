# Asymptotically Optimal Regret for Reinforcement Learning without Horizon Dependence

**Authors:** Runlong Zhou, Zihan Zhang, Maryam Fazel, Simon S. Du
**Published:** 2026-07-22 | **arXiv:** [2607.19854v1](http://arxiv.org/abs/2607.19854v1)

## Summary
This paper solves a fundamental problem in reinforcement learning: how to minimize regret (cumulative mistakes) without the regret depending on the planning horizon H. The authors propose a new algorithm achieving near-optimal regret bounds for tabular Markov decision processes — environments with discrete states and actions where the agent learns to maximize rewards. This matters because previous algorithms had regret that grew with log H, making them inefficient for long-term planning; this work removes that dependence entirely.

## Theoretical Background
The paper builds on reinforcement learning for Markov decision processes (MDPs) — a mathematical framework where an agent chooses actions in states to maximize cumulative rewards, and state transitions follow fixed probabilities. Key prior work includes regret analysis for contextual bandits (simpler problems where there's no sequential dependence between decisions) and earlier MDP algorithms by Zhang et al. (2021, 2022). The core technical challenge is that in time-homogeneous MDPs (where transition rules don't change over time), the optimal value functions — which measure expected future rewards — still vary with the remaining horizon H, creating a mismatch that standard analysis struggles to handle efficiently.

## Core Contributions
1. **Horizon-free regret bound**: Proves $\tilde{O}(\sqrt{SAK} + S^8A^3)$ regret without any dependence on H, matching the contextual-bandit lower bound up to logarithms. This completely removes the $\log H$ factor from prior work.

2. **Technical innovations**: (i) Exploits monotonicity of optimal value functions in horizon h to avoid a $\min\{\log H, S\}$ union-bound penalty; (ii) projects value functions onto an S-dimensional grid to compress the function class; (iii) introduces a horizon-truncation argument enabling reward-based exploration without separate exploration phases; (iv) designs a "cutting bonus" that maintains optimism while preserving monotonicity for planning.

3. **New theoretical tool**: Proves a bound on total deviation for time-homogeneous MDPs controlling variance with polynomial dependence on S but zero dependence on H.

## Practical Applications
- **Robot learning**: Enables safe long-horizon task learning (e.g., manipulation, navigation) where the planning window is unknown or variable.
- **Dialogue systems**: Horizon-free bounds help dialogue agents learn policies without specifying maximum conversation length in advance.
- **Game-playing agents**: In games with variable match lengths, this removes artificial horizon constraints from training algorithms.
- **Recommendation systems**: When the user session length is unpredictable, horizon-free regret guarantees make online learning more practical.

## Technical Limitations
- **Tabular assumption**: Requires discrete, small state-action spaces (S and A must be enumerable); fails for continuous or high-dimensional problems.
- **Polynomial dependence on S**: The $S^8A^3$ term becomes prohibitive for problems with even moderately large state spaces (e.g., S > 100).
- **Per-trajectory reward bound**: Assumes all trajectories yield rewards ≤ 1; real environments may require careful scaling or clipping.
- **Time-homogeneity**: Only applies to MDPs where transition dynamics don't change over time; doesn't handle non-stationary environments.
- **No function approximation**: Cannot leverage neural networks or other function classes to handle continuous/large problems.

## Open Questions
1. Can the $S^8A^3$ term be improved or eliminated? Is there a fundamental hardness barrier?
2. Does this horizon-free analysis extend to continuous state-action spaces with function approximation?
3. Can similar techniques apply to partial observability (POMDPs) or non-stationary MDPs?
4. How does this algorithm perform empirically against standard baselines (PPO, DQN) on benchmark tasks?
5. Is the monotonicity of value functions in h the key structural property, and can it be exploited in other settings (e.g., hierarchical RL)?

## Key Takeaway
New RL algorithm achieves optimal regret without depending on planning horizon, removing a theoretical bottleneck from prior methods.
