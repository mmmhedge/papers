# Forgetting Our Way to Shared Meaning: Effects of Forgetting on Conceptual Alignment in a Non-Partnership Coordination Game

**Authors:** Landon Liu, Mary Kelly, Alan Tsang
**Published:** 2026-07-13 | **arXiv:** [2607.11787v1](http://arxiv.org/abs/2607.11787v1)

## Summary
This paper investigates how memory and forgetting affect whether multiple agents can reach shared understanding of concepts through coordination. The authors model conceptual alignment as a non-partnership game (where agents don't have identical goals) and show that adaptive agents converge faster to agreement, while memory degradation—counterintuitively—can stabilize agreed-upon meanings. This matters because it reveals that perfect memory isn't necessary for language coordination, and that perceived convergence can diverge from actual agreement.

## Theoretical Background
The paper builds on two streams of work: (1) coordination games in game theory—formal models where agents benefit from aligning their behavior or understanding—and (2) models of semantic alignment and language emergence, which study how groups develop shared word meanings. Existing coordination game literature typically assumes partnerships (aligned payoffs), but this work extends into non-partnership settings where agents have divergent incentives. The concept of "shared meaning" refers to agents converging on similar semantic categories or interpretations of language. The authors use counterfactual simulations to isolate how memory properties (adaptiveness and degradation rates) affect convergence dynamics.

## Core Contributions
1. **Non-partnership coordination framework**: Extends classical coordination games to settings where agents don't share identical payoffs, making the model more realistic for real-world language emergence.
2. **Actual vs. perceived convergence**: Empirically demonstrates that agents can *perceive* they've aligned on meaning even when they haven't actually converged—a dissociation with practical implications.
3. **Memory degradation as stabilizer**: Shows that forgetting (memory decay) can produce *more stable* conceptual agreements than constant weighting of new information, challenging the intuition that perfect memory aids coordination.
4. **Adaptive learning effects**: Quantifies how adaptive agents (adjusting learning rates over time) outperform fixed-strategy agents in both speed and final alignment quality.

## Practical Applications
1. **Team communication protocols**: Understanding how teams with divergent goals (e.g., sales vs. engineering) naturally align on shared terminology despite conflicting incentives.
2. **Online community moderation**: Explains why communities develop stable jargon and norms even when members don't perfectly understand each other—useful for designing platform features that encourage healthy convergence.
3. **Multilingual dialogue systems**: Informs design of agents that must coordinate meaning across language barriers or when training data is incomplete or noisy.
4. **Negotiation and consensus-building**: Helps predict when groups will perceive false consensus and when genuine agreement breaks down.

## Product & Startup Ideas
1. **"Alignment Diagnostics" SaaS**: A tool that analyzes team communication logs to detect gaps between *perceived* and *actual* shared understanding, alerting managers when teams think they're aligned but aren't.
2. **Adaptive chatbot memory management**: A chatbot framework that strategically forgets low-value information over time to maintain stable, coherent personality and semantics—more human-like than perfect-recall systems.
3. **Negotiation coaching platform**: Real-time feedback during business negotiations or customer service interactions, signaling when parties are converging on false shared meaning versus genuine agreement.
4. **Cross-functional knowledge base**: Automatically detect when different departments (e.g., product vs. legal) are using the same terms differently and flag glossary conflicts before they cause costly misalignment.
5. **Game theory simulation studio**: A no-code tool for organizations to model coordination and language emergence in their specific multi-stakeholder settings, predicting convergence outcomes under different information policies.

## Technical Limitations
1. **Scalability**: Experiments likely use small agent populations; unclear how insights transfer to large communities (10s–1000s of agents) where network topology and information flow become critical.
2. **Simple semantic spaces**: The conceptual regions tested are probably low-dimensional; real language involves high-dimensional, ambiguous semantic structure that might exhibit very different memory effects.
3. **Synthetic task setup**: Non-partnership games in the paper likely have artificial payoff structures; real coordination (e.g., in teams) involves complex, implicit incentive structures not captured by formal game models.
4. **Memory models**: The memory degradation mechanisms (likely exponential or linear decay) are oversimplified compared to human forgetting curves and attentional dynamics in real conversation.
5. **No linguistic grounding**: The paper models abstract "conceptual regions" rather than actual language—how findings transfer to agents using natural language tokens or real semantic embeddings is unclear.
6. **Static agent populations**: Agents don't enter/exit; real communities have churn and newcomers who must learn existing conventions.

## Open Questions
1. **At what scale do findings break?** How does convergence behavior change with 100+ agents? Does network topology (e.g., hub-and-spoke vs. fully connected) matter?
2. **How do new agents learn?** When a new agent joins an aligned group, does perceived vs. actual convergence matter for their integration? Can they introduce conceptual drift?
3. **Can we measure these effects empirically?** Apply the framework to real team chat or community data—do humans show the same actual/perceived divergence?
4. **Optimal forgetting rates**: Is there a principled way to compute the *ideal* memory degradation rate for a given game structure, or is it task-dependent?
5. **Heterogeneous memory**: What if agents have *different* memory capabilities or forgetting rates? Does this help or harm convergence?
6. **Language evolution**: Does the framework predict realistic semantic drift and neologism formation, or is it too static to capture language change?

## Key Takeaway
Strategic forgetting helps multi-agent teams agree on shared meaning, even when they have conflicting goals.
