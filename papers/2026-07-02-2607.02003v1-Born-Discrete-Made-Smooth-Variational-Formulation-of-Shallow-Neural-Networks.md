# Born Discrete, Made Smooth: Variational Formulation of Shallow Neural Networks

**Authors:** Matej Benko, Pierre Bousquet, Iwona Chlebicka, Błażej Miasojedow
**Published:** 2026-07-02 | **arXiv:** [2607.02003v1](http://arxiv.org/abs/2607.02003v1)

## Summary

This paper replaces the discrete, non-convex optimization problem of training shallow neural networks with a continuous variational formulation over parameter densities. By casting the problem in weighted Sobolev spaces, the authors prove the resulting functional is λ-convex and globally well-posed — meaning the optimal solution can be found by solving a single linear system rather than running iterative gradient descent. This is significant because it provides theoretical guarantees and new algorithmic insights for a problem that has traditionally seemed intractable.

## Theoretical Background

Neural network training is conventionally viewed as minimizing a non-convex loss over discrete weight parameters — a problem with no global convergence guarantees. This paper builds on two modern theoretical perspectives:

1. **Mean-Field Theory**: Recent work (Mei et al., Chizat & Bach) models neural networks in the limit of infinite width by studying probability distributions over parameters, revealing connections to kernel methods and implicit bias.

2. **Neural Tangent Kernel (NTK)**: The observation that sufficiently over-parameterized networks behave like fixed kernel methods, with training dynamics governed by a linear operator.

3. **Wasserstein Geometry**: Prior work used optimal transport distances to study neural network limits, but faced regularity obstacles.

The key innovation here is replacing discrete parameters with a continuum of parameter densities and framing the problem using **elliptic regularity theory** — classical PDE machinery showing that solutions to certain elliptic equations inherit unexpected smoothness properties. By proving the loss functional is λ-convex (a generalization of strict convexity that allows negative curvature under controlled conditions), the authors unlock convex analysis tools normally unavailable for neural network problems.

## Core Contributions

1. **Variational Reformulation**: A continuous, well-posed functional over weighted Sobolev spaces that admits a unique global minimizer, replacing the non-convex discrete problem.

2. **Regularity Result**: Proof that the optimal parameter density achieves almost C³ regularity (three continuous derivatives) — substantially higher than Wasserstein or mean-field approaches typically guarantee.

3. **Linear System Solution**: Explicit construction showing the optimum solves a single linear elliptic PDE, eliminating the need for iterative optimization.

4. **Convergence Guarantees**: 
   - Generalization error scales as 1/α relative to regularization parameter α
   - Finite-width network of size N achieves continuum optimum with error O(1/N)

5. **Unified Framework**: Bridges NTK regime (kernels, no feature learning) and feature-learning regime by deriving both from the same variational principle.

## Connections to Other Work

This work provides a rigorous mathematical counterpart to [[Neural Tangent Kernel]] theory but avoids its limitation of requiring infinite width upfront. It improves upon [[Mean-Field]] approximations by achieving better regularity without Wasserstein geometry's discretization pitfalls.

The λ-convexity framework connects to broader convex analysis literature (Nesterov & Polyak) and offers an alternative to [[RLHF]]-style discrete optimization by providing a smooth, globally optimizable surrogate.

The result also relates to recent work on **neural network implicit bias** and **lottery ticket** phenomena — understanding which parameters matter — but from a density/continuum perspective rather than sparsity.

## Practical Applications

1. **Certified Neural Network Training**: For safety-critical domains (healthcare, autonomous vehicles), this provides a principled way to find provably optimal solutions to shallow networks without local minima traps.

2. **Inverse Problem Solving**: Parameter recovery in scientific computing (e.g., reconstructing physical coefficients from data) where theoretical guarantees matter.

3. **Regularization Design**: The explicit 1/α generalization bound informs principled choices of regularization strength, rather than cross-validation heuristics.

4. **Model Compression**: Understanding which parameter densities matter suggests which neurons to prune — grounded in variational optimality rather than magnitude heuristics.

5. **Uncertainty Quantification**: The well-posedness and regularity properties enable principled Bayesian inference over neural network parameters, with convergence guarantees.

## Product & Startup Ideas

1. **Verified Neural Network Library**: A toolbox (PyTorch/TensorFlow plugin) for shallow networks that solves the variational formulation exactly, marketed to regulated industries (finance, pharma) needing certified models.

2. **AutoML via Variational Inference**: A hyperparameter tuning service that uses the 1/α scaling law to automatically recommend regularization, competing with Bayesian optimization approaches but with theoretical backing.

3. **Physics-Informed Neural Networks (PINNs) Solver**: Specialized software for scientific computing that leverages the linear system solution to fit neural networks to PDEs with provable convergence guarantees.

4. **Neural Network Auditing Platform**: A service that formally verifies properties of trained shallow networks by reformulating them in the variational framework and checking mathematical invariants.

## Technical Limitations

1. **Shallow Networks Only**: The theory applies only to shallow (2-layer) networks; deep architectures remain a major open problem and likely require different mathematical machinery.

2. **Parameter Density Assumption**: The continuum limit assumes infinite-capacity parameter space; finite networks only approximate this at O(1/N) rate, which is slow for many practical regimes.

3. **No Computational Speedup Guaranteed**: Solving the linear system may be expensive in high dimensions; the paper doesn't address computational complexity of the PDE solver.

4. **Classification Limitations**: The theory focuses on regression; extension to classification (discrete labels, cross-entropy loss) is unclear.

5. **Regularity Assumptions**: The results assume data lies in appropriate Sobolev spaces; noisy, heavy-tailed, or adversarial data may violate these assumptions.

6. **Limited Architecture**: No nonlinear basis functions or alternative architectures explored; unclear how to generalize beyond ReLU-like activations.

## Open Questions

1. Can the variational framework extend to **deep networks** (3+ layers), or is shallow the fundamental limit of this approach?

2. What is the computational complexity of solving the resulting linear system in high dimensions? Is there a fast approximate solver?

3. Does the O(1/N) convergence rate match the minimax lower bound, or can it be improved?

4. How does the framework handle **implicit regularization** phenomena (e.g., early stopping, batch effects) observed empirically but absent in the continuum model?

5. Can the theory incorporate **data-dependent noise** (heteroscedastic errors) and still maintain global optimality?

6. Is there a natural way to extend this to **multi-task or federated learning** settings where parameter sharing across tasks matters?

7. How does the 1/α generalization bound compare empirically to standard practices (cross-validation, early stopping)?

## Key Takeaway

By reformulating shallow neural network training as a λ-convex variational problem over parameter densities, this paper transforms an intractable non-convex discrete optimization into a well-posed continuum problem solvable via a single linear system — providing the first global optimality guarantees for neural network training.
