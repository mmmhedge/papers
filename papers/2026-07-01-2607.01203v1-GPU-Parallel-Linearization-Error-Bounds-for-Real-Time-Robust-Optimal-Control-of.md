# GPU-Parallel Linearization Error Bounds for Real-Time Robust Optimal Control of Nonlinear and Neural Network Dynamics

**Authors:** Jeffrey Fang, Keyi Shen, Anutam Srinivasan, Glen Chou
**Published:** 2026-07-01 | **arXiv:** [2607.01203v1](http://arxiv.org/abs/2607.01203v1)

## Summary
This paper solves a critical problem in real-time control: when you use simplified linear approximations of complex nonlinear systems to make control problems tractable, how do you guarantee the actual system still respects safety constraints? The authors develop fast, mathematically verified bounds on linearization errors and embed them into a GPU-accelerated control solver, enabling robust feedback policies for neural network dynamics at real-time speeds (67 Hz on systems with 168 states).

## Theoretical Background
Real-time optimal control typically requires solving a planning problem repeatedly in a feedback loop. Nonlinear systems are hard to control optimally (the problem is generally non-convex), so practitioners approximate them with linear time-varying (LTV) models—which can be solved quickly using convex optimization. However, this approximation introduces error: the true nonlinear system may violate constraints that the linear plan respects. The paper builds on two traditions: (1) robust control theory, which uses uncertainty sets and reachability analysis to guarantee constraint satisfaction despite model mismatch, and (2) neural network verification, which computes certified bounds on NN outputs using relaxation techniques (affine bounds that over-approximate NN behavior). The key innovation is computing tight bounds on *how far* the linearization deviates from reality, then propagating these as formal disturbances through the dynamics using zonotopes (a geometric shape that tracks uncertainty compactly).

## Core Contributions
1. **Path-based Hessian bounds for analytic dynamics**: Tighter linearization error bounds than standard interval arithmetic methods by exploiting convexity properties along trajectories.
2. **Certified neural network linearization bounds**: Method to extract linearization error guarantees from NN verifier tools, using affine relaxations plus local Jacobian corrections—a practical bridge between NN verification and control.
3. **GPU-parallel robust control solver with extended capability**: Extended the system-level synthesis (SLS) framework to handle right-invertible disturbance matrices and non-zero-centered disturbance sets, enabling tighter zonotopic uncertainty propagation on GPU.
4. **End-to-end differentiable pipeline**: All bounds and propagation are differentiable, enabling online optimization of control policies that explicitly account for linearization error.

## Connections to Other Work
This work sits at the intersection of three areas: [[Neural Network Verification]] (using certified bounds to ensure properties), [[Robust Model Predictive Control]] (MPC under uncertainty), and [[System-Level Synthesis]] (a convex approach to feedback synthesis). It differs from classical robust control (which treats uncertainty as black-box disturbances) by computing the specific form of linearization error, making uncertainty sets much tighter. The use of zonotopes for uncertainty propagation relates to reachability analysis in hybrid systems. The neural network side connects to certified defenses in adversarial robustness, but applies verification to control rather than classification.

## Practical Applications
- **Autonomous vehicles**: Real-time trajectory planning around obstacles when the vehicle dynamics model is nonlinear (tire friction, aerodynamics).
- **Robotics with learned dynamics**: When a robot learns a neural network model of its own dynamics (e.g., from data), this provides guarantees that control laws derived from that NN model won't violate joint limits or collision constraints in reality.
- **Spacecraft attitude control**: Nonlinear rotational dynamics with formal safety guarantees for real-time guidance.
- **Quadrotor stabilization**: NN-based aerodynamic models can now be controlled with certified constraint satisfaction.
- **Industrial process control**: Nonlinear chemical reactors or thermal systems where NN models are fitted to data but must maintain product quality or safety limits.

## Technical Limitations
- **Scalability of NN verification**: The method relies on NN verifiers, which become expensive on very large networks; tight bounds may require smaller models than practitioners prefer.
- **Assumption of differentiability**: The Hessian bounds assume analytic dynamics or sufficiently smooth NN approximations; discontinuous or piecewise systems are not covered.
- **Zonotope explosion**: Uncertainty propagation over long horizons can cause zonotope complexity to grow exponentially, forcing approximation and potentially loosening guarantees.
- **Offline NN bounds**: NN relaxations are computed offline; the method cannot adapt if the NN is updated during deployment.
- **Limited to feedback policies**: The approach computes linear feedback laws; more expressive nonlinear policies (e.g., NN controllers) are not addressed.
- **Computational cost of online optimization**: While 67 Hz is real-time for many applications, it may be too slow for very high-frequency systems (e.g., microsecond timescales).

## Open Questions
- How do linearization error bounds degrade as system dimension increases further, and is there a scalability wall?
- Can this extend to stochastic dynamics (Brownian noise) rather than only deterministic disturbances?
- How sensitive are the formal guarantees to approximations made during zonotope propagation, and can we quantify loss of tightness?
- Can NN-based control policies (not just linear feedback) be certified using similar techniques, and would that close the conservatism gap further?
- How does this compare empirically to other recent certified control methods (e.g., those using barrier functions or Lyapunov-based approaches)?
- Can the method handle constraints that are themselves NN-based (e.g., learned safety regions)?

## Key Takeaway
By computing certified bounds on how much a linear control approximation differs from nonlinear reality, then propagating those bounds through robust control on GPU, this paper makes formally verified real-time optimal control practical for neural network dynamics.
