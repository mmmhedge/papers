# Stochastic Linear Bandits with Partially Observed Actions

**Authors:** Gautam Dasarathy, Vineet Gattani, Lalit Jain
**Published:** 2026-07-09 | **arXiv:** [2607.08971v1](http://arxiv.org/abs/2607.08971v1)

## Summary
This paper studies stochastic linear bandits — a fundamental sequential decision-making problem where an agent must learn which actions yield the best rewards — under a realistic constraint: the agent only observes partial information about each action. The key innovation is showing that when actions live in a low-dimensional subspace (intrinsic dimension), an algorithm called TOFU-POV can achieve near-optimal learning rates despite this missing information, solving a problem that was previously thought impossible in the general case.

## Theoretical Background
Stochastic linear bandits extend the classical multi-armed bandit problem: instead of choosing between discrete options, the agent selects action vectors and receives rewards that are linear functions of those vectors plus noise. The canonical algorithm for this setting is OFUL (Optimism in the Face of Uncertainty Linear bandit), which uses optimistic exploration to balance learning and earning rewards. The paper builds on linear bandit theory but introduces partial observability — the agent receives corrupted or incomplete action information — which fundamentally changes the learning problem. Prior work on bandits with missing data typically requires strong assumptions or accepts degraded regret bounds that scale with the ambient (full) dimension of actions, making learning prohibitively hard in high dimensions.

## Core Contributions
1. **Theoretical separation result**: Shows that partial observability does *not* make sublinear regret impossible when actions have low intrinsic dimension — overturning the intuition that missing information always causes linear (worst-case) regret.
2. **TOFU-POV algorithm**: A three-stage approach that (i) estimates the latent low-dimensional subspace from masked actions, (ii) uses a frozen representation to impute missing action coordinates in epochs, and (iii) applies OFUL in the recovered low-dimensional space. Achieves $\sqrt{T}$ regret scaling with intrinsic dimension $d^*$ rather than ambient dimension $d$.
3. **Tight characterization**: Provides regret bounds that explicitly quantify how missingness rate, decision set size, subspace conditioning, and intrinsic dimension interact — not just black-box dependence on these quantities.
4. **Rank-adaptive variant**: Proposes an algorithm that works without knowing the intrinsic dimension in advance.
5. **Novel lower bound**: A product construction that separates reward-learning uncertainty from missingness-dependent costs, establishing information-theoretic limits.

## Connections to Other Work
This work extends [[linear bandits]] and [[OFUL]] (Optimism in the Face of Uncertainty Linear bandit) to a partially observed setting. It relates to broader themes in [[reinforcement-learning]] about learning under partial observability (cf. POMDPs — Partially Observable Markov Decision Processes), and shares structure with dimensionality reduction techniques used in high-dimensional [[optimization]]. The low-intrinsic-dimension assumption echoes ideas in matrix completion and manifold learning — the agent must recover a hidden low-rank or low-dimensional structure from corrupted observations. The regret analysis borrows from recent advances in high-dimensional statistics and online learning under adversarial partial information.

## Practical Applications
1. **Recommendation systems**: When displaying items to users, platforms may only observe features the user actually sees (e.g., click-through signals on partial product descriptions) rather than full item attributes; TOFU-POV can optimize recommendations despite this.
2. **Healthcare / clinical trials**: Patient actions (treatment plans) have many features, but clinicians observe only a subset (e.g., symptoms that manifest, not all underlying biomarkers). Learning optimal treatment policies becomes tractable if treatments cluster in low-dimensional subspaces.
3. **E-commerce personalization**: Sellers observe only partial user interaction data; inferring user preferences and optimizing product recommendations requires learning from incomplete action descriptions.
4. **Ad targeting**: Ad creatives are high-dimensional, but platforms observe engagement metrics on only a subset of attributes. TOFU-POV enables efficient learning of targeting policies.

## Product & Startup Ideas
1. **Adaptive recommendation engine for healthcare platforms**: A service that learns optimal treatment recommendations by inferring latent clinical subspaces from partial patient data, reducing trial-and-error in personalized medicine.
2. **Bandit-powered A/B testing platform**: SaaS tool for e-commerce that handles missing/partial feature observations in multivariate testing, achieving faster convergence than standard methods when features are correlated.
3. **Privacy-preserving personalization**: A layer that sits between users and recommendation systems, learning user preferences while only observing random subsets of their behavioral data — selling differential privacy as a feature.
4. **Demand forecasting for sparse IoT sensor networks**: When sensors in supply chains fail or transmit incomplete state information, TOFU-POV helps forecast demand and optimize inventory from partial action/state observations.
5. **Portfolio optimization under incomplete market data**: For small investors or emerging markets with sparse price feeds, a tool that infers hidden market factors and optimizes asset allocation from partially observed price signals.

## Technical Limitations
1. **Intrinsic dimension must be small**: The algorithm's benefit hinges critically on the true intrinsic dimension being substantially smaller than ambient dimension. If it's not (or is unknown and the adaptive algorithm overestimates), regret degrades.
2. **Missingness pattern assumptions**: The analysis assumes missingness is random and uniform; structured or adversarial missing-data patterns may violate the theory.
3. **Epoch-wise frozen representation**: Imputing missing values using a fixed representation within epochs may lag if the subspace drifts; non-stationary or time-varying action distributions could break guarantees.
4. **Computational cost of subspace recovery**: Recovering the subspace via masked-action observations is statistically feasible but may be computationally expensive; scalability to very large action spaces is unclear.
5. **Linear reward model**: Restricts to linear bandits; non-linear rewards (e.g., with feature interactions) are not addressed.
6. **Conditioning and subspace identifiability**: Poor conditioning of the true subspace or ambiguity in recovering it from partial data can inflate regret constants.

## Open Questions
1. **Adaptive missingness**: Can the algorithm adapt to non-uniform or adversarial missingness patterns? What if the missingness rate itself depends on the agent's past choices?
2. **Computational lower bounds**: Are there hardness results (beyond information-theoretic regret bounds) showing when subspace recovery is computationally intractable?
3. **Non-linear rewards**: How does the framework extend to generalized linear bandits or neural bandit models with partial observations?
4. **Continuous/streaming subspace recovery**: Can the subspace be updated online rather than in frozen epochs, and does this improve regret?
5. **Correlated missingness**: What happens if missing coordinates are not independent — e.g., entire feature groups are hidden together?
6. **Finite-sample optimality**: Are the regret bounds tight up to constants, or is there room for improvement in the dependence on intrinsic dimension and conditioning?

## Key Takeaway
When actions are only partially observed, sublinear regret is still achievable if the actions lie in a low-intrinsic-dimension subspace — and TOFU-POV recovers this subspace and learns efficiently without knowing its dimension in advance.
