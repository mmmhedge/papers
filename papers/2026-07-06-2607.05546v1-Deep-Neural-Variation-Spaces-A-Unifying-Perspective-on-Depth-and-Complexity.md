# Deep Neural Variation Spaces: A Unifying Perspective on Depth and Complexity

**Authors:** Julia Nakhleh, Robert D. Nowak
**Published:** 2026-07-06 | **arXiv:** [2607.05546v1](http://arxiv.org/abs/2607.05546v1)

## Summary
This paper develops a unified mathematical framework for understanding deep neural networks by studying them as function spaces with controlled complexity. Rather than analyzing networks through parameter counts or specific activation types, the authors define recursive ℓ¹-bounded function spaces that work across many activation functions and depths, then prove that this notion of complexity actually limits what networks can represent—counterintuitively showing that depth may offer less advantage than commonly believed when complexity is properly controlled.

## Theoretical Background
The paper builds on functional analysis and approximation theory, particularly the concept of function spaces (mathematical sets of functions with defined complexity measures). Prior work on neural networks has relied on either homogeneity assumptions (activations that scale uniformly, like ReLU) or parameter-counting arguments that don't account for how complexity compounds across layers. The authors draw on variational characterization methods and norm-based complexity bounds—approaches that measure network capacity through mathematical norms (like ℓ¹ norms, which sum absolute values) rather than simply counting parameters. A key prior strand is representer theorems, classical results showing when optimal solutions to learning problems can be expressed as weighted combinations of data-dependent functions.

## Core Contributions
1. **Unified function space theory**: Defines a single recursive framework applicable to both homogeneous (ReLU) and non-homogeneous (sigmoid, tanh, etc.) activations, resolving a fragmentation in prior theory.

2. **Depth saturation result for ReLU**: Proves that in the univariate case, adding depth only rescales the function class by a constant factor—not exponentially—meaning depth doesn't create fundamentally new representable functions under complexity control.

3. **High-frequency limitation**: Shows that norm-constrained deep ReLU networks cannot represent high-frequency functions along any direction, revealing a fundamental trade-off between depth and oscillatory behavior.

4. **Novel representer theorem**: Establishes that optimal functions in these spaces have a concrete recursive structure, enabling constructive analysis.

5. **Qualitative smallness bounds**: Proves function classes remain structurally small (in a precise measure-theoretic sense) regardless of depth, contradicting naive "exponential expressivity" intuitions.

## Connections to Other Work
This work reframes long-standing debates about expressivity in [[deep neural networks]]. It relates to [[Neural Tangent Kernel]] theory and infinite-width limits, but avoids those asymptotic regimes. The depth-vs-complexity tension echoes [[universal approximation]] debates and [[lottery ticket hypothesis]] findings. The ℓ¹ complexity norm framework connects to sparsity-promoting methods and [[compressed sensing]], where ℓ¹ norms encourage sparse solutions. The paper implicitly challenges narratives from work on depth-based expressivity (e.g., proving exponential separation between shallow and deep nets) by showing those gains vanish under appropriate norm constraints rather than parameter constraints.

## Practical Applications
1. **Network design and pruning**: Guides practitioners toward using ℓ¹-norm constraints as principled alternatives to L2 regularization or dropout, predicting which depth increases will actually help.

2. **Regularization strategies**: Informs hyperparameter selection—suggests that for smooth function learning, shallow networks with controlled norms may outperform deep ones.

3. **Generalization bounds**: Provides tighter, architecture-aware complexity measures for statistical learning theory, improving test-set performance prediction.

4. **Activation function selection**: Shows that homogeneous vs. non-homogeneous activation choice matters less than norm control, enabling flexible architecture design.

5. **Scientific machine learning**: For physics-informed neural networks (PINNs), the inability to represent high-frequency functions under norm control is directly relevant to learning smooth physical solutions.

## Product & Startup Ideas
1. **Automated network design tool**: Software that recommends network depth, width, and regularization strength given a target function class and complexity budget, using this theory to avoid wasteful overparameterization.

2. **Certification service for neural networks**: Provide formal guarantees on smoothness and frequency content of learned functions, critical for safety-critical domains (autonomous driving, medical imaging).

3. **Energy-efficient ML platform**: Exploit the depth saturation result to build shallower, faster networks for edge devices, with theoretical guarantees on expressivity.

4. **Interpretability layer**: Build tools that visualize the recursive function space structure, helping practitioners understand what a trained network can actually represent.

5. **Complexity-aware AutoML**: Extend AutoML systems to optimize explicitly over function space complexity rather than parameter count, targeting specific applications (time series with bounded spectral content, smooth regression).

## Technical Limitations
1. **Univariate vs. multivariate gap**: Depth saturation is proven rigorously only for univariate ReLU; the multivariate case relies on weaker asymptotic arguments. The generalization may not hold sharply in high dimensions.

2. **Practical optimization unclear**: The theory characterizes *representable* functions but says little about what gradient descent actually learns. The recursive ℓ¹-bounded construction may not be implicit in standard training dynamics.

3. **Activation-specific results limited**: While the framework is general, the strongest results (depth saturation, high-frequency bounds) apply mainly to ReLU. Mileage varies for other activations.

4. **Norm constraints impractical**: ℓ¹ bounds on layer weights are less natural to enforce than L2 penalties during training. The theory doesn't clarify how to implement the bounds efficiently.

5. **Static analysis**: The theory treats networks as static function classes. It doesn't address the dynamics of learning, overfitting, or adaptation to data distribution.

## Open Questions
1. Does depth saturation hold sharply in high dimensions, or only asymptotically? Can you construct explicit multivariate functions that require exponential depth under ℓ¹ norm constraints?

2. What do neural networks *actually learn* under ℓ¹ constraints during training? Do standard optimizers implicitly satisfy the bounds, or do explicit penalties degrade generalization?

3. Can this framework explain phenomena like grokking, double descent, or benign overfitting observed in practice?

4. How does the theory extend to convolutional, recurrent, or attention-based architectures? Do analogous depth saturation results hold?

5. Is there a constructive algorithm that, given a target function and complexity budget, builds a shallow network that provably represents it? Can we tractably compute the minimal depth required?

## Key Takeaway
Depth loses its representational advantage for neural networks once you control complexity with the right norm—suggesting that much of deep learning's empirical success may come from regularization and optimization dynamics rather than fundamental architectural depth.
