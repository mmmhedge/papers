# Differentiable Logic Gate Networks for Low-Latency EEG Classification on Edge Devices

**Authors:** Shyamal Y. Dharia, Stephen D. Smith, Camilo E. Valderrama
**Published:** 2026-07-20 | **arXiv:** [2607.18149v1](http://arxiv.org/abs/2607.18149v1)

## Summary
This paper proposes Differentiable Logic Gate Networks (Diff-Logic), a neural architecture that compiles into pure Boolean circuits executable via bitwise operations, enabling real-time EEG classification on power-constrained edge devices. The approach outperforms floating-point neural networks on dementia detection and achieves 2.9× speedup over standard MLPs while maintaining nearly constant inference time across 10× increases in model size—solving a critical bottleneck in portable brain-computer interfaces.

## Theoretical Background
Real-time neural inference on edge devices (small, battery-powered computers like the Nvidia Jetson Nano) is hampered by floating-point arithmetic, which is slow and energy-intensive on resource-constrained processors. Existing approaches include binarized neural networks (BNNs — networks with weights and activations quantized to ±1) and quantization techniques, which reduce precision but still rely on expensive arithmetic. Diff-Logic takes a more radical approach: it represents the entire network as a Boolean function computable entirely with bitwise CPU operations (AND, OR, NOT), which are orders of magnitude faster than floating-point math. The key insight is that logic gates can be made differentiable during training via smooth approximations, allowing end-to-end learning while the final model compiles to pure combinatorial logic.

## Core Contributions
1. **Diff-Logic architecture**: A neural network design that is natively compilable to Boolean circuits, eliminating floating-point computation entirely.
2. **Iso-parameter experimental validation**: Rigorous comparison against MLP and BNN baselines at matched parameter counts (50k–500k) across two distinct EEG tasks—a strong experimental design that isolates the effect of the architecture itself.
3. **Scalability proof**: Demonstration that Diff-Logic inference latency remains nearly constant as model complexity grows 10×, whereas conventional networks scale linearly—a striking property.
4. **Dementia classification benchmark**: 80.2% Macro F1 on binary dementia detection, outperforming the MLP baseline by 6.8 percentage points, suggesting logic-based models may be particularly suited to certain medical EEG tasks.

## Practical Applications
1. **Portable dementia screening devices**: Hospital-grade EEG classification on wearable headsets that run for days on a battery, enabling early diagnosis in remote or resource-limited settings.
2. **Real-time emotion recognition for accessibility**: Emotion detection in augmentative communication devices for locked-in patients or motor-impaired users, where latency directly impacts user experience.
3. **Embedded seizure detection**: Continuous EEG monitoring on implanted or wearable devices that must operate for months without recharging.
4. **Edge-deployed sleep stage classification**: On-device sleep monitoring in consumer wearables (smartwatches, rings) for sleep health tracking without cloud offloading.
5. **Low-power medical telemetry**: Remote patient monitoring in field hospitals or ambulances where power draw and weight are severe constraints.

## Technical Limitations
1. **Task-specificity**: The performance advantage appears tied to specific EEG tasks; emotion recognition showed MLP superior, suggesting Diff-Logic works best on certain problem geometries (unclear which a priori).
2. **Limited task diversity**: Only two EEG tasks tested; generalizability to other biomedical signals (ECG, EMG) or non-medical domains unknown.
3. **Training overhead**: The paper does not report training time or computational cost; differentiable logic approximations may require expensive gradient computation during learning.
4. **Boolean circuit explainability claims**: While logic gates are theoretically more interpretable, a 500k-parameter Boolean network is likely as inscrutable as a conventional network in practice.
5. **Hardware assumptions**: Speedup depends heavily on CPU bitwise operation efficiency; benefits may not transfer to GPUs or specialized accelerators where floating-point is already fast.
6. **Scalability ceiling**: Unclear whether the constant-latency property holds at million+ parameter scales or whether model quality degrades with scale.

## Open Questions
1. **Why does Diff-Logic outperform on dementia but not emotion?** Is there a theoretical characterization of which classification geometries favor Boolean circuits?
2. **How does training time and convergence speed compare?** Does the differentiable approximation slow learning?
3. **Can Diff-Logic transfer learn?** Can models pre-trained on one task/dataset adapt to another on-device?
4. **Scaling laws**: What is the scaling relationship between model size and accuracy? Does Diff-Logic obey similar power laws as conventional networks?
5. **Multi-modal fusion**: Can Diff-Logic models efficiently combine EEG with other signals (ECG, accelerometer) on the same device?
6. **Formal verification**: Can the compiled Boolean circuits be formally verified for safety-critical medical applications?

## Key Takeaway
Logic-based neural networks enable sub-millisecond EEG classification on tiny batteries, outpacing floating-point models while scaling to large architectures without latency growth.
